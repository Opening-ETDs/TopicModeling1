Local project deploy: 
    1. install the flask environment
    2. PLEASE change path of dataset from line 42 to line 50. 
    3. source venv/bin/activate
    4. export FLASK_APP=ETD
    5. flask run
    6. change the path of dataset in App.java
    7. start the java project and wait for dataset indexing
    6. WEB uri: "serverIP:5000"
